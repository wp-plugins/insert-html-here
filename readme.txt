== Description ==

Allows one or more blocks of html code to be inserted into a Wordpress post or page.  This plugin should work with Wordpress 2.5.0 or greater.  I have only tested it on my site running Wordpress 2.6.3.

== Installation ==

1. Upload `insert-html-here.php` to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. The InsertHTMLHere admin page found in the Wordpress settings allows for the html blocks of code to be entered.
1. Place `[insert-html-here x]` in your post or page where x is an integer corresponding to a specific InsertHTMLHere block of html.
